# Axionix — Sistema Operativo Empresarial

> **v3.2** · Automatiza y escala tu negocio con Axionix.

Axionix es una plataforma empresarial y sistema de Punto de Venta (POS) en la nube, diseñada para adaptarse desde comercios locales hasta operaciones corporativas. Tres divisiones unificadas bajo un mismo núcleo: autenticación, base de datos, permisos granulares y sincronización en tiempo real.

## 🧩 Divisiones del producto

### 01 · Axionix Core — SaaS
Automatización para negocios basados en servicio: clínicas, barberías, restaurantes y spas. Incluye agenda inteligente, expediente digital, flujos operativos y reportes en una sola vista.

### 02 · Axionix POINT — POS
Sistema de Punto de Venta ágil y hardware-friendly para refaccionarias, retail y tortillerías. Búsqueda instantánea, inventario multi-sucursal, facturación y cortes de caja en segundos.

### 03 · Enterprise — Proyectos a la medida
Desarrollo enterprise con equipo dedicado, SLA definido y roadmap compartido. Orientado a industria, refacciones y logística.

## 🚀 Desarrollo local

Clona el repositorio e inicia un servidor local (no se requiere build step):

```bash
git clone https://github.com/rodrigost1455-hub/axionix.git
cd axionix

# Con cualquier servidor estático, por ejemplo:
npx serve .
# o
python -m http.server 3000
```

Luego abre `http://localhost:3000/index.html` en tu navegador.
